// TYPES.XML TOOLS
Types.xml Editor
9 tools for managing, editing, and generating types.xml files. Paste your XML or upload a file, make your changes, and copy the result.
🔍 Extractor 📋 Builder ⏱ Restock & Lifetime 🔢 Nominal Converter ⚖ Nominal Adjuster 0️⃣ Zero Nominal 🧹 Tier Remover 🚗 Vehicle Lifetime ✂ Splitter
🔍
Classname Extractor
Extract all classnames from a types.xml file. Returns a clean list you can copy or use in other tools.
📋
Classname → Types Builder
Paste a list of classnames (one per line) and generate a complete types.xml block with custom values.
⏱
Restock & Lifetime Editor
Batch update restock and/or lifetime values across an entire types.xml. Leave a field blank to skip updating it.
🔢
Nominal Converter
Set ALL nominal values in a types.xml to a specific number in one click.
⚖
Nominal Adjuster
Increment or decrement all nominal values by a set amount. Use negative numbers to decrease.
0️⃣
Zero Nominal Updater
Find all entries with nominal=0 and set them to a new value. Optionally exclude specific classnames.
🧹
Tier Remover
Strip all Tier1/Tier2/Tier3/Tier4 value tags from a types.xml file.
🚗
Vehicle Lifetime Editor
Update lifetime values for vehicle entries in types.xml. Detects vehicle classnames automatically.
Auto-detection: Looks for classnames containing common vehicle keywords: Car, Truck, Bus, Heli, Boat, Bike, UAZ, Sedan, Offroad, Transit, Olga, Gunter, Sarka, Skoda, Lada, Humvee, HMMWV.
✂
Types Splitter
Split a large types.xml into separate files grouped by classname prefix. Each group is shown separately so you can copy them individually.
How it works: Groups entries by the first N characters of the classname. E.g. with length 3: AK74, AKM → "AK_types.xml", M4A1, MP5 → "M4_types.xml" etc.
