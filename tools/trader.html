Trader Config Tools
Convert classnames and configs between trader formats. Supports TraderPlus and Dr. Jones Trader.
💱 TraderPlus Converter 🔄 Dr. Jones → TraderPlus 🔧 Vehicle Parts Config 🏪 Expansion Market ✏️ Expansion Editor 🔍 Expansion Extractor
💱
TraderPlus Converter
Convert a list of classnames to TraderPlus price config format. One classname per line.
Format: Generates TraderPlus product lines in the format: ClassName, BuyPrice, SellPrice, MaxStock
🔄
Dr. Jones → TraderPlus Converter
Convert a Dr. Jones TraderObjects config to TraderPlus format. Paste your existing trader config below.
Supported input: Dr. Jones TraderObjects XML format with Item elements containing ClassName, SellPricePercent, MaxStockThreshold attributes.
🔧
Vehicle Parts Config
Generate a TraderPlus vehicle config with common parts automatically based on the vehicle classname.
Auto-generates: Wheel, Hood, Door, Trunk, Engine, Battery, Spark Plug entries using the vehicle classname as prefix.
