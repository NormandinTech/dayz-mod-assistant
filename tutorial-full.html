08Chat Area
💬Message RowsChat panel
Each message shows a line number, author (YOU / DZMod.AI), role label (USER / ASSISTANT), and timestamp. Luna's responses stream in real time.
📋Code BlocksInside messages
Code is syntax-highlighted with a language label. Hover to reveal a copy button in the top-right corner. Shortcut for last code block: Ctrl+Shift+C.
👍Feedback ButtonsBelow each response
Thumbs up/down appear after each Luna response. Feedback is logged anonymously to help improve Luna over time.
💡Follow-up SuggestionsBelow each response
4 context-aware follow-up chips appear after each response. They change based on what topic was discussed. Click any to instantly send that follow-up. Examples: "Show me a complete example", "How do I test this?", "What are common mistakes?"
💾Auto-saveBackground
Chat history is saved to browser local storage after every message. Close and return — the full conversation is restored with line numbers and timestamps intact.

10Uploading Your Mod Files

Upload your actual mod files and Luna reads the real content. This is the most powerful way to use Luna.

Supported: .c .cpp .h .hpp .xml .json .txt .cfg .sqf .ext .bisurf .cs .ini

1
Click "+ Upload files"
Select one or more files. Multiple files supported.
2
Files appear as orange tags
Green dot = saved to local storage and persists across sessions.
3
Ask about your files
"Review my types.xml for errors", "Why isn't my modded class firing?", "Check my config.cpp."
4
Remove when done
Click ✕ on any tag to stop Luna referencing that file.
⚠
Size limit: Files truncated at 6,000 chars each. For large files, paste only the relevant section into a .txt file.
11PBO Inspector

Built-in PBO parser. Drop any .pbo file and inspect it — no external tools needed. Runs entirely in your browser, nothing uploaded to any server.

1
Click "Load PBO"
Select a .pbo file. Parsed client-side instantly.
2
Browse the file tree
Text files listed by folder. Binary files (.p3d, .paa, .wss) skipped automatically.
3
Decompiled configs
Binarized config.bin files auto-decompile to readable .cpp format, shown with a green "decompiled" badge.
4
Select files
Use "Select all text", "Deselect all", or "Scripts only" to filter. Count updates in real time.
5
Click "Add to Context"
Files load as orange tags. Ask Luna about the mod's code, structure, and configs.
💡
Great for reverse-engineering mods. Load a PBO, add scripts to context, then ask "explain what this mod does" or "how does this vehicle class work?"
⚠
Encrypted PBOs: Some authors encrypt their PBOs. If config.bin shows "could not be decompiled", the content is protected and unreadable.
12Community Contribute

The Contribute button shares your uploaded files with the shared community knowledge base, helping Luna give better answers to all users over time.

1
Upload files first
Use "+ Upload files" or "Load PBO" to add files.
2
Click Contribute
All uploaded files are chunked and stored in the shared knowledge base.
3
Review results
Luna reports chunks added per file. Files contributed within 24 hours are skipped (cooldown).
⚠
Only contribute your own files. Do not contribute other people's mod files without permission.

17FAQ
Does Luna remember previous sessions?
Chat history is saved in browser local storage and restored on return. Luna's AI context is limited to the last 20 messages per session — very long old conversations may be partially trimmed.
What DayZ version is Luna tuned for?
Current stable DayZ PC release (1.25). She knows the current Enforce Script API, CE format, and DayZ Tools workflow.
Can Luna write a complete mod for me?
Yes — break it into parts. Ask for config.cpp first, then scripts, then economy configs. Luna writes complete working code, never truncated.
Does Luna support console DayZ?
No. Console DayZ doesn't support mods. Luna is PC-only via Steam Workshop.
Is my uploaded code private?
Files are stored only in your browser and sent to the AI for processing. Nothing is stored on any server unless you click Contribute.
Why does Luna sometimes get things wrong?
She's very good but not perfect. Always test code suggestions on a local server before deploying. If something looks off, paste your code and ask her to review it directly.
My license key isn't working.
Paste the full key with no extra spaces. If it still fails, email vegabotgg@outlook.com with your Gumroad order number.
Can I use Luna on mobile?
Yes — the UI adapts to small screens. The sidebar collapses to a horizontal scroll row and some tab buttons hide. Core chat works fully on mobile.
How do I restart the onboarding tour?
Open browser console (F12) and type restartTour() then press Enter.
