How do I override OnEnterVehicle in EnforceScript?
L
Override it in your modded class in 4_World layer:
modded class CarScript {
  override void OnEnterVehicle(
    Human player, int seatIdx
  ) {
    super.OnEnterVehicle(player, seatIdx);
    // your code here
  }
}
S
How do I check which seat they entered?
L
Use seatIdx === 0 for the driver seat. Passengers are index 1+.
